# BhoomiAI (भूमिAI) 🌾
### *Empowering the Roots, Engineering the Future.*

BhoomiAI (formerly Digital Sarathi) is a next-generation, AI-driven agricultural and logistical ecosystem designed to bridge the gap between advanced technology and ground-level implementation. It serves as a comprehensive digital companion for farmers while providing a robust framework for national resource management.

---

## 🚀 The Vision: Beyond Agriculture
While BhoomiAI starts in the fields, its ultimate objective is **National Resource Sovereignty**. 

By integrating state-of-the-art AI with local infrastructure, BhoomiAI aims to become the **single unified application for the Government to handle the entire nation’s ration and supply chain**. Through this platform, organizing national-scale logistics becomes simpler, more transparent, and highly efficient, ensuring that no grain is wasted and no citizen is left behind.

---

## 🛠️ Advanced Technologies Used

BhoomiAI leverages a cutting-edge tech stack to ensure high performance, low latency, and maximum accessibility:

### 🧠 Intelligence & Language
- **LangGraph & LangChain**: Powering a sophisticated agentic workflow that maintains context across complex agricultural consultations.
- **RAG (Retrieval-Augmented Generation)**: Utilizing **FAISS** and **Sentence Transformers** to provide ground-truth answers from verified government documents and pest advisories.
- **Groq LPU™**: Harnessing ultra-fast LLM inference for real-time, human-like interaction.
- **Faster-Whisper**: High-accuracy Speech-to-Text (STT) for hands-free operation in the field.
- **Piper TTS & Edge-TTS**: Natural-sounding, localized Text-to-Speech (TTS) for multi-dialect support.

### 💻 Core Infrastructure
- **FastAPI (Python)**: A modern, high-performance backend designed for asynchronous task handling.
- **React + Vite + Tailwind CSS**: A lightning-fast, responsive, and aesthetically polished frontend.
- **Recharts**: Interactive data visualization for farm yield and market trend analysis.
- **MySQL**: Relational database for robust farmer profiles and historical record-keeping.

## 🌍 Impact & Problem Grounding

### The Challenge
- **Information Asymmetry**: Farmers often lack access to scientific crop planning or real-time pest warnings.
- **Language Barriers**: Most advanced AI tools are restricted to English, alienating millions of rural users.
- **Logistical Complexity**: Government ration distribution (PDS) often suffers from "middle-man" inefficiencies and lack of real-time tracking.

### The BhoomiAI Solution
- **Localized Wisdom**: AI that speaks the farmer's language and understands local soil (NPK) and weather patterns.
- **Scientific Planning**: Generates optimized crop plans based on budget, land size, and risk appetite.
- **Transparent Logistics**: A blueprint for government-scale organization, moving from fragmented systems to a single, unified "Command Center" application.

## 🛠️ Setup & Installation

### Backend
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Configure `.env` with your API keys (Groq, OpenAI, etc.).
4. Run the server: `python main.py`.

### Frontend
1. Navigate to the `frontend` directory.
2. Install packages: `npm install`.
3. Start the dev server: `npm run dev`.
